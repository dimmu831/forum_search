# 多平台討論區搜尋工具

一個簡單而強大的網頁應用程式，讓您同時搜尋 Dcard、PTT 和 Mobile01 三個討論區的內容。

## 功能特性

- 🔍 **多平台搜尋**：同時搜尋 Dcard、PTT 和 Mobile01
- 📊 **表格展示**：搜尋結果以清晰的表格格式呈現
- 🎨 **美觀設計**：現代化的紫色漸層設計
- ⚡ **快速響應**：實時搜尋結果
- 📱 **響應式設計**：支援桌面和行動設備

## 使用方法

1. 在搜尋框中輸入您想搜尋的關鍵字
2. 點擊「搜尋」按鈕
3. 系統會為三個討論區各生成一個 Google 搜尋連結
4. 點擊連結可直接查看該討論區的搜尋結果

## 技術棧

- **前端**：HTML5 + CSS3 + JavaScript
- **後端**：Node.js + Express
- **搜尋方式**：Google 站內搜尋 (site: 操作符)

## 安裝與運行

npm install
npm start
http://localhost:3000

## API 端點

POST /api/search - 搜尋多平台討論區
